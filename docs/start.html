Start

Three steps. Steal it tonight.

1. Clone the kit

git clone https://github.com/mirror-factory/openbook.git
cd openbook

2. Load the skill

Point your coding agent at this folder and load skills/openbook. Agents without skill support: paste the short fallback in USE.md.

3. End the run with a report

Invoke /openbook (or ask for the handoff). The agent writes REPORT.md as a Brief by default. Then check structure:

python scripts/check_report.py path/to/REPORT.md

Optional warm-paper PDF: render_report.py. Full install notes for PDF live in the README.
