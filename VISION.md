# OpenBook

**Agent work, written for humans.**

&nbsp;

The agent worked all night.

It owes you a story, not a log.

&nbsp;

Trust is not a vibe. Trust is a document shape:

a story you can skim,
decisions you can answer,
receipts you can check.

&nbsp;

So the contract is small.

**Sixty seconds** tells you what happened.
**Decisions** ask plainly, one at a time, with room for a pen.
**Receipts** wait underneath, for whenever you doubt.

&nbsp;

Dead ends are content.
Length is earned, never padded.
The editor may cut. Never invent.

&nbsp;

Not a framework. Not a platform. Not a login.

Markdown in a folder. Steal it tonight.

&nbsp;

Agents will keep getting longer horizons.
Humans will keep having the same morning.

This is how those two facts stop fighting.

&nbsp;

Don't trust the agent.

**Read the OpenBook.**

&nbsp;
